# Imagens do Projetta v6

Coloque aqui as imagens do sistema. O `index.html` referencia elas por nome.

## Mapeamento das imagens

| Arquivo | Onde é usado | Descrição |
|---------|-------------|-----------|
| img-1.png | Tela de login | Logo de erro |
| img-2.jpg | Proposta comercial | Fundo página 1 (comprimir com tinyjpg.com) |
| img-3.jpg | Proposta comercial | Capa da proposta (comprimir com tinyjpg.com) |
| img-4.jpg a img-27.jpg | Catálogo de modelos | Fotos dos modelos de portas |

## Como adicionar imagens

1. Salve a imagem nesta pasta com o nome correto (ex: img-2.jpg)
2. Faça commit e push
3. O sistema carrega automaticamente

## IMPORTANTE
- **NÃO** converta imagens para Base64
- **NÃO** cole imagens dentro do HTML
- Mantenha as imagens **sempre** nesta pasta como arquivos separados
- Use https://tinyjpg.com para comprimir imagens grandes
